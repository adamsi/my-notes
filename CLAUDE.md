# CLAUDE.md

* Next.js App Router + TypeScript + Tailwind CSS + shadcn/ui.
* Prefer Server Components; use Client Components only when required.
* Keep components small, reusable, and strongly typed.
* Keep code simple— avoid overengineering and unnecessary dependencies.
* Mobile-first, accessible, and responsive.
* Use consistent spacing, large touch targets, and clean minimalist UI.
* Always provide loading, empty, and error states.
* Remove unused code and keep the codebase clean.
* You are allowed to modify .env files, never commit them.
* After every feature/ refactor commit push with clear and short commit message.
